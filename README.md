# 🗞️ tabnews-cli

A CLI client to [TabNews](https://tabnews.com.br).

📦 Install dependences

```bash
bun install
```

🔧 Start dev

```bash
bun run dev
```

🔥 Start

```bash
bun run start
```

or

```bash
npm run start
```

This project was created using `bun init`, with bun v1.0.23. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
